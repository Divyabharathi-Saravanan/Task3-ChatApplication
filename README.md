# Chat Application

A simple chat application implemented in Java using sockets and multithreading.

## Project Structure
* `ChatServer.java`: The server-side code that handles client connections and broadcasts messages.
* `ChatClient.java`: The client-side code that connects to the server and sends/receives messages.

## How to Run
1. Compile the server code: `javac ChatServer.java`
2. Run the server: `java ChatServer`
3. Compile the client code: `javac ChatClient.java`
4. Run the client: `java ChatClient`

## Features
* Multiple clients can connect to the server and chat with each other.
* Messages are broadcasted to all connected clients.

## Technologies Used
* Java
* Sockets
* Multithreading
